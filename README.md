# Hexterika Cyberlab — Website

The service website for Hexterika Cyberlab (HC), an independent IT &
cybersecurity micro-studio offering small, well-scoped, deliverable-based
services (vulnerability assessments, AWS IAM setup, backup & recovery).

## Static site hosted on GitHub Pages

## Live pages

+ `index.html` — landing page
+ `hc-services.html` — Hexterika Cyberlab services

## Assets

`assets/css/` — base.css (shared), index.css, hc-services.css
`assets/images/profile/` — site imagery

## Archived

/archived/ holds pages from the earlier "Hexterika Breaks Free" version of the
site (portfolio, certificates, skills, about, and various freelance-service
pages). They are unlinked from the live site and kept for reference/reuse only.
Their internal links and styling are preserved best-effort and are not
maintained.
