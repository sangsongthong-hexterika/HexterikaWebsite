Projects

These GitHub repositories are collections of my write-ups, walkthroughs, and mock reports based on hands-on cybersecurity lab challenges. The content comes from platforms like TryHackMe, PortSwigger Web Security Academy, and my own virtual home lab setups, including an Active Directory lab.

Each repository is organized by topic and documentation style. For example, the TryHackMe Walkthroughs repo contains step-by-step guides written after completing specific rooms, while the PortSwigger repo focuses on web security labs. The Mock Penetration Testing Reports repo includes report-style documentation based on scenarios built from lab exercises.

Some collections like the IT-Cybersecurity Oneshot repo include one-off topics I've encountered, such as command-line tricks or security concepts that support my broader learning. These are snapshots of knowledge I've picked up, whether or not I plan to expand on them immediately.

All repositories are self-contained, with individual README files explaining each project's purpose. While some topics are still in progress, these projects reflect my real learning journey and will continue to grow and improve over time. New additions aim to be more structured and refined, even if older ones remain as-is for historical context.
